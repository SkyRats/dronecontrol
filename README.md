# dronecontrol

:robot: Control Nodes for PX4 compatible drones

### Requirements
* ROS
* MAVROS

### Launching mavros
* via USB telemetry `roslaunch mavros px4.launch fcu_url:="/dev/ttyUSB0:57600"`
* simulation `roslaunch mavros px4.launch fcu_url:="udp://:14550@127.0.0.1:14557"`
